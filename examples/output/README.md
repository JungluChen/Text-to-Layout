# Example output — IDC capacitor

Reference artifacts generated from [`../idc_0p6pf.json`](../idc_0p6pf.json) (a
~0.6 pF interdigital capacitor, 24 finger pairs on M1). Committed as a sample of
what the plugin produces; verification status was **pass** (9/9 checks).

| File | What |
|---|---|
| `idc.gds` | GDSII (KLayout-verified: top cell `IDC_*`, layer 1/0 = M1, 50 shapes, bbox 286 × 304 µm) |
| `idc.svg` | 2-D preview |
| `idc.json` | Lossless Geometry IR (layers, polygons, bbox, ports, metadata) |

## Regenerate

```bash
# CLI
textlayout generate examples/idc_0p6pf.json --out examples/output

# or via the API
curl -s -X POST localhost:8000/layout/generate \
  -H 'Content-Type: application/json' \
  -d @examples/idc_0p6pf.json
```

> Note: the GDS top-cell name carries a random suffix per build (to avoid
> gdsfactory's process-global cell-name collisions), so regenerated bytes differ
> while the geometry is identical.
