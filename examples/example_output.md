# Example Output

```powershell
py -3 -m uv run python skills\text-to-gds\scripts\text_to_gds_tool.py toolchain --output-name manhattan_jj.gds --jc-ua-per-um2 2.0
```

## Deliberate DRC Failure Example

The default barrier is `0.22 um` wide. Raising the min-width threshold to
`0.3 um` should fail:

```powershell
py -3 -m uv run python skills\text-to-gds\scripts\text_to_gds_tool.py drc workspace\artifacts\manhattan_jj.gds --min-width-um 0.3
```

Representative violation:

```json
{
  "rule": "min_bbox_width",
  "message": "Shape minimum bounding-box dimension 0.22 um is below 0.3 um.",
  "severity": "error",
  "layer": [4, 0],
  "min_dimension_um": 0.22
}
```
