# Reproducibility Checklist

- Pin the package, PDK, process-run, solver, and dependency versions.
- Record GDS, process, netlist, simulation, and measurement hashes.
- Preserve raw data, units, calibration, uncertainty, random seeds, and commands.
- Report skipped external backends explicitly; never convert missing evidence into a pass.
- Run DRC, LVS, convergence, passivity, reciprocity, causality, and quantum-limit checks.
